1. What does the man command do? Type in man rm. How do you scroll and get out?
A) The man command displays the manual of a command. 

2. Look at the man page for ls. What does the -l flag do? What does the -a flag do?
A) -l uses a long listing format, -a makes the command not ignore hidden files.

3. How do you jump between words in the terminal?
A) CTRL + Left/Right

4. How do you get to the end of a line in terminal?
A) CTRL + e

5. How do you move your cursor to the beginning in terminal?
A) CTRL + a

6. How do you delete a word (without pressing backspace multiple times) in terminal?
A) CTRL + w (while curser is on the word you want to delete.)

7. What is the difference between a terminal and shell?
A) The shell executes commands, the terminal is an interface for the shell.

8. What is an absolute path?
A) The absolute path of a file is the directory starting from root.

9. What is an relative path?
A) The relative path starts at the current working location.

10. What is a flag? Give three examples of flags you have used.
A) A flag is an extension to a command to edit the way that command works. (-l, -a, -r)

11. What do the r and f flags do with the rm command?
A) The r flag will recursively delete files/folders, it is used to delete a folder that contains files. The f flag forces the deletion.
